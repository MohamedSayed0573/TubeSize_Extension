# Privacy Policy for TubeSize

Last updated: March 6, 2026

TubeSize is a browser extension that shows estimated file sizes for YouTube video streams.

TubeSize does not require an account, does not use ads or analytics, and does not sell personal data.

## What TubeSize Uses

TubeSize may access:

- The current YouTube watch page URL, only to extract the video ID.
- Video metadata and stream size information available on the page.
- Your extension settings and cached results stored with Chrome extension storage.

This data is used only to show video sizes, save preferences, and cache results.

## Backup Server

TubeSize mainly works locally in your browser. If local extraction fails and you enable `Use backup server`, the extension may send the YouTube video ID to our backup server at `https://api.mohammedsayed.dev`.

Like most web services, the server may also receive technical request data such as IP address, user agent, and request timing. This is used only to serve the request, keep the service reliable, and prevent abuse.

The backup server code is open source and included in this GitHub repository.

## Storage and Retention

- Settings and cached results stay in your browser until they expire, are replaced, or you clear them.
- Server logs, if any, are kept only as long as needed for reliability, security, and abuse prevention.

## Your Choices

- Disable `Use backup server` at any time.
- Clear cached data from the Options page.
- Remove the extension to stop future data processing by the extension.

## Changes

This policy may be updated from time to time. The latest version will be published at the same public URL used for the extension listing.
